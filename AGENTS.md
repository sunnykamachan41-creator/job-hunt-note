# AGENTS.md

## Project Goal

Build a personal job hunting tracker web app.

## Core Concepts

* Main UI is a horizontal timeline (gantt-like)
* Prioritize overview and readability
* Avoid card-heavy UI
* Hide details unless clicked
* Google Calendar is helper only

## Technical Constraints

* Use Next.js + TypeScript + Tailwind
* Use Google Sheets as database
* Use Google Calendar API for one-way sync
* Do NOT use Supabase, Firebase, Prisma

## Data Structure

Main sheets:

* companies
* events
* settings

## Important Rule

If implementation details are unclear, ask questions before coding.
